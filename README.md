# 💰 Controlador de Gastos Fijos

Aplicación web simple y elegante para gestionar y controlar tus gastos mensuales fijos.

## ✨ Características

- 📝 **Gestión de Gastos**: Agrega, edita y elimina conceptos de gastos
- 💳 **Datos de Pago**: Guarda CBU, alias, números de cuenta o cualquier información de pago
- 💵 **Cálculo Automático**: Total mensual actualizado en tiempo real
- 💾 **Persistencia Local**: Todos los datos se guardan automáticamente en localStorage
- 📱 **Diseño Responsivo**: Funciona perfecto en móviles, tablets y desktop
- 🎨 **Interfaz Moderna**: Diseño limpio con Tailwind CSS

## 🚀 Uso

1. Abre `index.html` en tu navegador
2. Agrega tus conceptos de gastos (alquiler, servicios, suscripciones, etc.)
3. Completa los datos de pago para cada concepto
4. Ingresa los montos mensuales
5. ¡Tu información se guarda automáticamente!

## 💡 Ejemplos de Uso

- **Vivienda**: Alquiler, luz, gas, agua, internet
- **Suscripciones**: Netflix, Spotify, gimnasio, software
- **Servicios**: Seguros, celular, transporte
- **Otros**: Educación, préstamos, cuotas

## 🔧 Tecnologías

- HTML5
- JavaScript (Vanilla)
- Tailwind CSS
- localStorage API

## 📦 Instalación

No requiere instalación. Solo descarga el archivo y ábrelo en tu navegador:

```bash
git clone https://github.com/MartuTahir/CONTROLADOR_GASTOS.git
cd CONTROLADOR_GASTOS
# Abre index.html en tu navegador
```

## 🗑️ Borrar Datos

Para borrar todos los datos guardados, abre la consola del navegador (F12) y ejecuta:

```javascript
localStorage.clear()
```
